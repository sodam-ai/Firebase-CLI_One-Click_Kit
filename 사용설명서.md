## 0. 시작 전에 (딱 한 가지)

폴더 안의 **`시작하기.bat`** 를 더블클릭하세요.
파란 글씨로 **내 컴퓨터 상태**와 **"지금 할 일"** 이 한 줄로 나옵니다. 그 안내만 따라가면 됩니다.

> 검은 창(또는 파란 창)이 떠도 괜찮습니다. 이건 정상입니다.

**메뉴 고르는 법:** **위/아래 화살표(↑↓)로 움직여 Enter**, 또는 **숫자 키**를 누르면 됩니다.
초록색 **"지금 이것! ←"** 가 다음에 할 일입니다. 빨간색 **(주의)** 는 조심할 항목(제거)입니다.

## 0-1. 파일이 안 열리거나 "보호됨" 경고가 뜰 때 (다운로드 차단 풀기)

인터넷에서 받은 파일은 윈도우가 안전을 위해 **잠가둘 때**가 있어요. 풀어주면 됩니다.

- **방법 1 (가장 쉬움):** 압축을 풀기 **전에**, 받은 zip 파일을 **우클릭 → 속성 → 맨 아래 '차단 해제(Unblock)' 체크 → 확인**. 그다음 압축을 푸세요.
- **방법 2:** `시작하기.bat`를 **우클릭 → 속성 → '차단 해제' 체크 → 확인** 한 뒤 더블클릭.
- 파란 **"Windows의 PC 보호"** 창이 뜨면 → **추가 정보 → 실행**.

> 한 번 `시작하기.bat`가 실행되면, 폴더 안 나머지 파일의 잠금은 **자동으로 풀립니다.** (그래서 보통 처음 한 번만 신경 쓰면 됩니다.)

## 1. 설치하기 (처음 한 번만)

1. `시작하기.bat` → 화면에서 **[1] 설치하기** 선택 (화살표로 고르고 Enter, 또는 숫자 `1`).
2. (이제 **관리자 허용 창은 보통 안 뜹니다.** Node.js가 없어 자동 설치할 때만 한 번 뜰 수 있고, 그땐 **[예]**.)
3. 검은 창이 영어로 진행됩니다. **닫지 말고 기다리세요** (1~3분).
4. `ALL DONE! INSTALL WAS A SUCCESS!` 가 보이면 성공입니다.

**Node.js(필수 부품)가 없다고 나오면?**
→ "자동으로 설치할까요? (Y/N)" 가 나옵니다. **`Y`** 를 누르면 알아서 깔아줍니다.
→ 자동 설치가 끝나면 창을 닫고 **`시작하기.bat`를 다시 한 번** 실행하세요.

---

## 2. 처음 사용 — 로그인 (구글 계정 연결)

1. `시작하기.bat` → **[2] 사용하기**
2. 메뉴에서 **1) Login to Firebase** (로그인) 입력
3. 인터넷 브라우저가 저절로 열립니다 → **내 구글 계정으로 로그인**
4. "성공" 화면이 나오면 창을 닫고 검은 창으로 돌아오세요.

> 로그인은 한 번만 하면 됩니다. 다음부터는 기억합니다.

---

## 3. 내 프로젝트 준비 (배포하려면 먼저)

순서가 중요합니다: **로그인 → 프로젝트 만들기 → 배포**

1. 메뉴 **9) Make a new Firebase project here** (프로젝트 만들기)
2. 영어로 질문이 나오면 **방향키(↑↓)와 스페이스바**로 고르고 **Enter**
3. 메뉴 **10**으로 내 프로젝트 목록 확인, **11**로 사용할 프로젝트 선택

---

## 4. 인터넷에 올리기 (배포 / Deploy) — 주의!

> 배포 = 내 앱을 **모두가 볼 수 있게 인터넷에 공개**하는 일입니다.

1. 메뉴 **15) Deploy everything** (전체 배포)
2. 안전을 위해 **`YES`(대문자)** 를 입력해야 진행됩니다.
3. 프로젝트 설정(`firebase.json`)이 없으면, 키트가 **막고** "먼저 9번을 하세요" 라고 알려줍니다.

---

## 5. 제거하기 (지우기)

1. `시작하기.bat` → **[3] 제거하기** (빨간 **(주의)** 표시)
2. **`YES`** 입력 → 설정 백업할지 묻습니다(Y/N).
3. 깨끗이 지워집니다.

**안심하세요:** 제거해도
- 내가 만든 **코드 파일**은 그대로
- **Node.js**도 그대로
- 구글 서버에 있는 **데이터**도 그대로 둡니다.

---

## 6. 오류가 났을 때 (당황하지 마세요)

| 이런 화면이 나오면 | 이렇게 하세요 |
|---|---|
| 파일이 안 열림 / **"Windows의 PC 보호 / 차단됨"** | 위 **0-1. 다운로드 차단 풀기** 참고 (우클릭→속성→차단 해제) |
| `firebase ... not found` (설치했는데 못 찾음) | 창을 모두 닫고 **새 창**에서 다시 시작하기.bat 실행 |
| `Can not reach the internet` | 와이파이/인터넷 확인 후 다시 |
| `Install did not work` | 같은 폴더의 **`firebase_install.log`** 열어서 마지막 줄 확인 (백신·회사망 차단이 흔함) |
| 메뉴에서 엉뚱한 번호 입력 | 그냥 다시 메뉴로 돌아갑니다. 걱정 마세요 |

## 7. 자주 묻는 질문

**Q. 명령어를 외워야 하나요?**
A. 아니요. 번호만 누르면 됩니다. (고급 사용자는 RUN 메뉴 25번에서 직접 명령 입력 가능)

**Q. 돈이 드나요?**
A. 이 키트는 무료입니다. 단, Firebase 서비스 사용량이 많아지면 구글 요금이 생길 수 있어 구글 콘솔에서 확인하세요. (이 키트는 요금과 무관합니다.)

**Q. 잘못 눌러서 망가지면요?**
A. 위험한 작업(배포·제거)은 모두 **한 번 더 물어봅니다.** 그냥 닫아도 됩니다.
